# Точка Силы → неподписанный .ipa

Готовый Capacitor-проект: веб-код лежит в `www/`, при сборке нативная iOS-часть (`ios/`) генерируется прямо в GitHub Actions, ничего вручную ставить не нужно.

## Как собрать

1. Создай новый репозиторий на GitHub (например `tochka-sily-ios`).
2. Загрузи туда содержимое этого zip **как есть** — с папками `.github/`, `www/`, файлами `package.json`, `capacitor.config.json`, `.gitignore`.
3. Зайди во вкладку **Actions** репозитория. Если экшены не включились сами — включи их (кнопка сверху).
4. Либо просто сделай `git push` в ветку `main` — workflow `Build unsigned IPA` запустится сам. Либо запусти вручную: Actions → Build unsigned IPA → Run workflow.
5. Сборка идёт на `macos-14` раннере ~5–10 минут: ставит зависимости → `npx cap add ios` → `npx cap sync` → архивирует Xcode-проект без подписи → упаковывает `Payload/App.app` в `.ipa`.
6. По завершении зайди в конкретный run → внизу раздел **Artifacts** → скачай `TochkaSily-unsigned-ipa` (это zip с самим `.ipa` внутри).

## Дальше

Скачанный `.ipa` — на Sideloadly как обычно: перетаскиваешь файл, вводишь свой Apple ID, жмёшь Start. Sideloadly сам переподпишет и поставит на телефон.

## Если сборка упадёт на шаге xcodebuild

Версии Xcode/Capacitor меняются, и флаги отключения подписи иногда ведут себя по-разному между релизами Xcode. Если `xcodebuild archive` с `CODE_SIGNING_ALLOWED=NO` откажется собираться — открой лог шага в Actions, обычно там прямо написано, чего не хватает (чаще всего — версия SDK/схемы или ругань на entitlements). Скинь мне текст ошибки, поправим workflow под конкретную версию раннера.

## Что можно донастроить

- **appId** в `capacitor.config.json` (`ru.kelwin.tochkasily`) — можно сменить на свой, ни на что не влияет при sideload через Sideloadly.
- **Иконка/сплэш** — Capacitor подставит дефолтную. Если нужна своя — кинь мне PNG 1024×1024, добавлю генерацию иконок через `@capacitor/assets` в workflow.
- Каждый пуш в `main` пересобирает `.ipa` заново — обновил `www/` (например, поправил `app.js`) → закоммитил → новый билд в Artifacts.
